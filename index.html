What is facewiki?

Facewiki is an open wiki for everyone and everything, Where you can create your own wiki page and ask anyone to write about you.

Why to use facewiki?

Wiki allows you to create your own page for anything as long as it makes sense. We are on a mission to create wiki which can be used to create pages without restrictions.

Vision

We want to create a centralised system for all kind of information which will be helpful to access from anywhere by anyone.
